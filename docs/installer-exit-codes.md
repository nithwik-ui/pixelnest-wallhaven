# PexelNest Installer Exit Codes

This document describes the installer exit codes used by the PexelNest Windows installer.

| Exit Code | Meaning |
|-----------|---------|
| 0 | Installation completed successfully |
| 1602 | Installation cancelled by user |
| 1618 | Another installation is already in progress |
| 1638 | Application already installed |
| 112 | Disk full |
| 3010 | Installation completed, reboot required |
| 1625 | Installation blocked by system security policy |
